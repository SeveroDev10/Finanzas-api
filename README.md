# 💰 Finanzas Fáciles

Aplicación de escritorio desarrollada en Python para la gestión completa de finanzas personales.

## 🚀 Funcionalidades

- Sistema de usuarios con autenticación segura (hash SHA-256)
- Registro de ingresos y gastos
- Edición y eliminación de movimientos
- Categorías personalizadas
- Presupuestos con alertas automáticas
- Recordatorios de pagos
- Exportación a CSV y TXT
- Visualización de datos con gráficas

## 🧪 Tecnologías

- Python
- CustomTkinter (interfaz gráfica)
- Matplotlib (gráficas)
