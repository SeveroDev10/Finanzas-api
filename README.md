# 💰 Personal Finance Manager

Aplicación desarrollada en Python para gestionar ingresos y gastos de manera sencilla, permitiendo al usuario llevar un control claro de sus finanzas personales.

## ✨ Funcionalidades

- Registro de ingresos y gastos
- Balance automático
- Control mensual
- Edición de movimientos
- Interfaz visual amigable
- Resumen financiero

## 🧪 Tecnologías utilizadas

- Python
- (Tkinter,customtkinter)
